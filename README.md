## 실행 방법(과정)
### 일반 실행
1. 프로젝트 폴더 경로 내에서 아래의 명령어를 실행한다.
```bash
python3 bot.py
```

### Run
1. 프로젝트 폴더 경로 내에서 아래의 명령어를 실행한다.
```bash
./run.sh
```

### Stop
1. 프로젝트 폴더 경로 내에서 아래의 명령어를 실행한다.
```bash
./stop.sh
```
